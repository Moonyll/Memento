Memento

1. Les Variables :

  • Pour déclarer une variable entière a : int a = valeur.
  • Pour afficher la variable a : Utiliser Console.WriteLine(a).
  • Pour enregistrer une variable renseignée dans la console : var = Console.ReadLine().
  • Pour mettre la console en attente : Utiliser Console.ReadKey().
  • Pour réinitialiser la fenêtre console : Utiliser Console.Clear().
  • Pour changer la couleur du texte : Utiliser Console.ForegroundColor = Console.Color.Blue,...,Red.
  • Pour changer la couleur du fond : Utiliser Console.BackgroundColor = Console.Color.Blue,...,Red.
  • Pour changer la postion du texte : Utiliser Console.SetCursorPosition(n,k) = Marge de n (gauche) , Marge de k (haut).
  • Pour afficher la date d'aujourd'hui en version courte : DateTime.UtcNow.Date .ToShortDateString().
  • Opérations sur des variables numériques : Utiliser les opérateurs +, -, *, /.
  • Opérations mathématiques : Utiliser Math.XXX(). Ex : Math.Pow(a,n) pour avoir la puissance n de a.
  • Pour incrémenter / décrémenter une variable numérique : Utiliser ++ ou --.
  • Pour concatener des éléments : Utiliser le symbole $.
  • Pour convertir une chaine de caratctère : Utiliser int.Parse().
  • Rappel des différents types de variables :
    • byte = Entier de 0 à 255.
    • short = Entier de -32768 à 32767.
    • int = Entier de -2147483648 à 2147483647.
    • long = Entier de -9223372036854775808 à 9223372036854775807.
    • float = Nombre simple précision de -3,402823e38 à 3,402823e38.
    • double = Nombre double précision de -1,79769313486232e308 à 1,79769313486232e308.
    • decimal = Nombre décimal convenant particulièrement aux calculs financiers.
    • char = Représente un caractère.
    • string = Une chaine de caractère.
    • bool = Une valeur booléenne (vrai ou faux).

2. Les Tableaux - Les Listes :

  • Pour déclarer un tableau d'entiers : int[] = new int []{n1,...,ni}.
  • Pour déclarer un tableau de chaines de caractères : string [] = new string []{"srting 1",...,"string i"}.
  • Attention ! L'indice i diffère de valeur. Valeur = tableau[indice i].
  • La longueur d'un tableau est : tableau.Length.
  • Pour déclarer une liste d'entiers : liste <'int> = new list<'int>().
  • Pour déclarer une liste de chaine de caractère : liste <'string> = new list<'string>().
  • Pour ajouter des éléments : liste.Add(Elément 1)...liste.Add(Elément n).
  • Pour insérer des éléments : liste.Insert(Indice i,Elément i)...liste.Insert(Indice n,Elément n).
  • Pour créer une énumération : Dans la partie Class avant Static Void : enum name {enum 1,...,enum n}.
  • Pour afficher les indices i d'une énumération : Enum.GetValues(typeof(name)).
  • Pour afficher les valeurs d'une énumération : Méthode N°1 ---> Enum.GetNames(typeof(name)) [GetName aussi].
  • Pour afficher les valeurs d'une énumération : Méthode N°2 ---> name el 1 = name.enum 1.
  • Exemple a. : Console.WriteLine(Enum.GetName(typeof(name),indice i) = affiche valeur de l'indice i.
  • Exemple b. : Console.WriteLine(Enum.GetNames(typeof(name).GetValue(indice i))) = affiche valeur de l'indice i.

3. Les Conditions

  • If{} , Else{} & Else If{}.
  • If{Condition True} & Else{Condtion False}.
  • Les différents opérateurs et compléments sur les opérations :
    • == Egal.
    • <= Inférieur ou Egal.
    • < Strictement Inférieur.
    • >= Supérieur ou Egal.
    • > Strictement Supérieur.
    • && Et.
    • || Ou (il existe aussi ^).
    • ! Non Egal (Not).
    • % Reste d'une division.
    • i += k équivaut à : i = i + k et i++ (pour k=1).
    • i -= k équivaut à : i = i - k et i-- (pour k=1).
    • i *= k équivaut à : i = i * k.
    • i /= k équivaut à : i = i / k.
    • i %= k équivaut à : i = i % k.
  • La fonction "Switch" permet d'alléger le codage au bout de 3 conditions :
  • switch(variable) {case(variable i) : condition i; break;... case(variable n) : contition n; break; default : default condition; break;}

4. Les Boucles

  • Boucle While : int i = 0; while(condition) {action; i+=k;}.
  • Boucle Do While : int i = 0; do {action; i+=k;} while(condition).
  • Boucle For : (initialisation (nombre = k); condition (nombre < n); incrémentation (nombre++)) {action;}.
  • Boucle Foreach : foreach (var name in tab){action;}.
  • .Take(n) permet par exemple de récupérer les n premiers éléments d'un tableau tab[] : foreach (int number in tab.Take(n)) {action;}
